## About This Portfolio

This is a personal portfolio and research showcase built with [Quarto](https://quarto.org), designed to present my academic work, statistical projects, and professional qualifications in a modern, aesthetically pleasing interface. It features a warm terracotta and cream design system with smooth scroll-reveal animations, a click-to-expand lightbox for images, inline PDF viewers, and dark mode support — no generic blue templates here.

The site is hosted on **GitHub Pages** and renders directly from the `/docs` output folder generated by Quarto.

## Getting Started

### Prerequisites

- [R](https://cran.r-project.org/) (≥ 4.2.0)
- [RStudio](https://posit.co/download/rstudio-desktop/) (recommended)
- [Quarto](https://quarto.org/docs/get-started/) (≥ 1.4.0)

### Clone the Repository

```bash
git clone https://github.com/odeymofokeng-wits/my_portfolio.git
cd my_portfolio
```

### Render the Site

**From R console:**

```r
install.packages("quarto")  # if not already installed
quarto::quarto_render()
```

**From terminal:**

```bash
quarto render
```

**Live preview with hot-reload:**

```bash
quarto preview
```

This starts a local development server. Open the URL shown in your terminal to see the site with live updates as you edit.

---

## Deploying to GitHub Pages

The site builds to the `docs/` directory. To publish:

1. Push the full project to your GitHub repository
2. Navigate to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose `main` branch and `/docs` folder
5. Click **Save**

Your portfolio will be live at:

```
https://odeymofokeng-wits.github.io/my_portfolio/
```

To update: edit `.qmd` files → run `quarto render` → commit & push the updated `docs/` folder.

## Customisation Guide

| What to Change | Where to Edit |
|----------------|---------------|
| Colours & palette | CSS variables at the top of `styles.css` + SCSS files |
| Page content | `.qmd` files (Markdown + HTML) |
| Navigation links | `navbar` section in `_quarto.yml` |
| Footer content | `page-footer` section in `_quarto.yml` |
| Fonts | Google Fonts import in `_quarto.yml` `include-in-header` |
| Animations | `styles.css` — `.reveal`, `@keyframes` sections |
| Project file URLs | `projects.qmd` — update local `assets/` paths |

## Features

- **Scroll-reveal animations** — sections fade in as you scroll using Intersection Observer
- **Image lightbox** — click any image to view full-size with animated zoom and backdrop blur
- **Inline PDF viewer** — research posters and reports display directly in the page
- **Staggered card animations** — project cards animate in sequence
- **Dark mode** — toggle between cream and espresso themes via Quarto's built-in switcher
- **Responsive** — optimised for mobile, tablet, and desktop
- **Grain texture** — subtle SVG noise overlay for warmth and depth
